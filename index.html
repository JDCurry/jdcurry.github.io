Key Technologies in Emergency Management

Satellite Networks

  • Examples: Starlink, GOES satellites, GPS systems
  • Use Cases: Global communication, weather monitoring, navigation
  • Pros: Global coverage, resilient infrastructure
  • Cons: Weather interference, latency considerations

Cybersecurity & Networking

  • Examples: Cisco networking, firewalls, encryption protocols
  • Use Cases: Infrastructure protection, secure communications
  • Pros: Enhanced security, standardized protocols
  • Cons: Ongoing maintenance, complexity

Warning Systems & Social Media

  • Examples: IPAWS, social media monitoring tools
  • Use Cases: Public alerts, situational awareness
  • Pros: Rapid dissemination, wide reach
  • Cons: Information verification challenges

Expertise Focus: AI/LLM in Grant Management

Current Project: Streamlining Federal Grant Management

I'm developing an innovative solution using machine learning models (GPT-2 Small and GPT-Neo 1.3B) to streamline the federal grant application process for emergency management departments. This project focuses on FEMA's Hazard Mitigation Grant Program (HMGP) and Building Resilient Infrastructure and Communities (BRIC) programs.
